🛠️ Especificaciones Tecnicas del ADN Angoleño
Este contrato, libre de caracteres especiales para maxima interoperabilidad, mide la consistencia fisica y el origen cultural de las tecnicas de Luanda y sus provincias:

Nivel de Viscosidad (uint256): Escala que mide la fluidez mecanica del Funge. A diferencia del Fufu ghanes (que busca elasticidad), el Funge angoleño se calibra en un nivel 9 para asegurar una textura cremosa y sin grumos.

Usa Aceite de Palma (bool): Validador de la base lipidica esencial para platos como el Muamba, donde el aceite actua como conductor de sabor y color carmesi.

Es Fusion Lusa (bool): Identificador de platos que integran tecnicas de la gastronomia portuguesa (como el arroz de cabidela) con ingredientes y contextos locales.

Sistema de Feedback: Funciones de darLike y darDislike para la participacion de la comunidad.
